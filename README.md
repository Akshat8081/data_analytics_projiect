
#Project Title
# Retail Sales Data Cleaning using Microsoft Excel


## Project Overview

.This is my first Data Analytics project.

.The objective of this project was to clean a retail sales dataset using Microsoft Excel before performing any analysis.

.Data cleaning is one of the most important steps in data analytics because accurate insights depend on clean and reliable data.


## Dataset Information

- Total Records: 1,200
- Total Columns: 14
- Tool Used: Microsoft Excel


## Data Cleaning Steps

The following cleaning steps were performed:

- Checked all column data types.
- Verified date format consistency.
- Identified missing values in the CouponCode column.
- Preserved missing CouponCode values because they may represent orders where no coupon was used.
- Verified numerical columns for proper formatting.
- Ensured dataset consistency before analysis.



## Excel Functions Used

- TYPE()
- Filter
- Sort
- Data Validation
- Basic Data Cleaning Techniques
