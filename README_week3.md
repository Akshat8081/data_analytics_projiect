# 📖 SQL Concepts Covered

## SELECT

Retrieved required columns and complete records from the dataset.

Example:

```sql
SELECT * FROM retail_sales;
```

---

## WHERE

Filtered records based on specific conditions.

Examples:

```sql
SELECT *
FROM retail_sales
WHERE TotalPrice > 2000;
```

```sql
SELECT *
FROM retail_sales
WHERE PaymentMethod = 'Credit Card';
```

---

## ORDER BY

Sorted records in ascending and descending order.

Example:

```sql
SELECT *
FROM retail_sales
ORDER BY TotalPrice DESC;
```

---

## GROUP BY

Grouped data into categories for summary analysis.

Example:

```sql
SELECT Product,
       SUM(TotalPrice) AS Revenue
FROM retail_sales
GROUP BY Product;
```

---

## Aggregate Functions

Used SQL aggregate functions:

- COUNT()
- SUM()
- AVG()

Examples:

```sql
SELECT COUNT(*) AS TotalOrders
FROM retail_sales;
```

```sql
SELECT SUM(TotalPrice) AS TotalRevenue
FROM retail_sales;
```

```sql
SELECT AVG(TotalPrice) AS AverageOrderValue
FROM retail_sales;
```
